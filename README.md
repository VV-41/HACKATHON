# NyayaAI ⚖️

NyayaAI is an AI-powered legal assistant that helps users understand legal queries using Natural Language Processing.

## Features
- Legal question answering
- TF-IDF based NLP search
- Flask backend
- Interactive frontend

## Tech Stack
Python  
Flask  
Scikit-learn  
HTML  
CSS  
JavaScript  

## Run the Project

pip install -r requirements.txt  
python app.py  

Open browser:

http://127.0.0.1:5000
